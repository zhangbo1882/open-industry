# 三花智控（SZ002050 / 02050）

## 公司概况

浙江三花智能控制股份有限公司，全球领先的热管理控制系统零部件供应商。以热泵技术和热管理系统研发应用为核心，业务横跨**制冷空调电器零部件**和**汽车零部件**两大板块，并向**仿生机器人机电执行器**等新兴领域拓展。

A+H两地上市（2025年6月H股上市）。全球6大研发中心、8大主要生产基地，销售网络遍布中国、亚太、欧洲、北美。

## 业务结构

### 1. 制冷空调电器零部件（成熟压舱石，59.9%收入）

**全球家用/商用空调、商用制冷、工业制冷关键供应商。**

| 产品 | 说明 |
|------|------|
| 电子膨胀阀 | 制冷系统流量精确控制 |
| 四通换向阀 | 热泵系统制冷/制热切换 |
| 截止阀 / 电磁阀 / 球阀 | 流体控制阀门系列 |
| 微通道换热器 | 高效紧凑式换热器 |
| Omega 泵 | 高效泵类产品 |
| 控制器 | 系统控制单元 |

应用：空调、冰箱、工业制冷、冷链运输、热泵采暖、洗衣机等。

### 2. 汽车零部件（增长主引擎，40.1%收入）

**全球新能源汽车热管理系统零部件领先供应商。**

| 产品 | 应用场景 |
|------|----------|
| 电子膨胀阀 | 座舱热管理 / 电池热管理 |
| 电子水泵 | 电池冷却 / 电机冷却循环 |
| 电子水阀 | 多回路热管理切换 |
| 板式换热器 | 制冷剂-冷却液换热 |
| 集成组件 | 多合一热管理模块 |

已进入**大部分头部新能源车企供应链**（含 Tesla），覆盖座舱/电池/电机电控全场景热管理。

### 3. 第三增长曲线（培育中）

| 方向 | 状态 |
|------|------|
| **仿生机器人机电执行器** | 战略拓展，基于热管理+机电技术积累 |
| 储能热管理 | 新增长点 |
| 数据中心热管理 | 新增长点 |
| 传感器 | 新增长点 |

2025年定为公司"AI应用元年"，全面推进AI与研发/制造/管理深度融合。

## 竞争格局

### 全球热管理赛道

| 维度 | 格局 |
|------|------|
| 制冷空调阀件 | 全球龙头，电子膨胀阀/四通换向阀份额领先 |
| 汽车热管理 | 全球领先，与**电装、翰昂、法雷奥**等竞争，深度绑定新能源车企 |
| 仿生机器人执行器 | 新进入者，借助机电技术积累 + Tesla合作基础切入 |

### 壁垒与优势

1. **技术积累深厚**：热管理技术跨制冷/汽车/机器人三领域复用
2. **客户绑定深**：已进入大部分头部新能源车企，份额稳定
3. **全球化制造**：6研发+8生产基地，贴近客户产能布局
4. **A+H双平台**：拓宽融资渠道，支撑国际并购和产能扩张
5. **Tesla生态**：汽车热管理 → 机器人执行器的供应链延伸逻辑

## 人形机器人布局

> 三花智控在 Morgan Stanley Humanoid 100 中被归入 **Body（身体）— 综合自动化/执行器模组**，与西门子、霍尼韦尔、罗克韦尔等并列，是中国关键的人形机器人执行器供应商。

- 基于汽车热管理电子膨胀阀/电子水泵的**精密流体控制 + 电机驱动**技术积累
- 向**仿生机器人机电执行器**方向拓展
- 董事长致辞明确提到"打造第三增长曲线，向仿生机器人机电执行器等领域积极拓展"
- Tesla Optimus 供应链身份使其在机器人执行器领域具有先发优势

## 风险提示

> [!warning] 第三曲线尚未贡献收入
> 仿生机器人执行器业务仍处于战略投入期，尚无独立收入披露，短期不贡献利润。

> [!warning] 汽车业务增速放缓风险
> 汽车零部件2025年增速9.1%，低于2024年增速，新能源车渗透率到达高位后增速可能进一步放缓。

> [!warning] 海外经营风险
> 海外收入占43%，面临地缘政治、关税、汇率波动风险。2025年外汇差额亏损扩大。
