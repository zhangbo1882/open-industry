# 深科技（SZ000021）

## 公司概况

深圳长城开发科技股份有限公司，全球领先的专业电子制造企业，EMS行业深耕超40年。实际控制人为**中国电子信息产业集团有限公司**（中国电子）。构建了**存储半导体、高端制造、计量智能终端**三大主营业务格局。

## 业务结构

### 1. 存储半导体（40.91亿，25.98%，毛利率20.40%）

**国内高端存储芯片封测龙头**，深圳沛顿+合肥沛顿双基地运营。

| 产品线 | 说明 |
|--------|------|
| DRAM封测 | DDR4/DDR5、LPDDR4/LPDDR5 |
| NAND封测 | UFS3.1/UFS4.1、uMCP |
| 先进封装 | NAND 32D超高堆叠、GDDR多芯片倒装、Panel FO超薄封装 |
| HDD盘基片 | 企业级大容量存储基片 |
| HDD磁头 | 受客户产能转移影响，2025年销售量同比下滑 |

- 成立**先进封装研究院**，与高校合作设立先进制造技术创新中心
- 核心材料国产化突破（粘接膜、基板、镀钯铜线）
- 深圳沛顿获广东省制造业单项冠军，合肥沛顿获国家专精特新"小巨人"

### 2. 高端制造（85.35亿，54.20%，毛利率9.71%）

**EMS电子制造服务**，一站式电子产品制造服务。

| 应用领域 | 说明 |
|----------|------|
| 医疗健康 | 高潜领域，客户结构优化 |
| 汽车电子 | 新能源汽车竞争激烈，增速放缓 |
| 消费电子 | 业务量稳步提升 |
| 智能家居/IoT | 持续拓展 |
| 高精密注塑件 | 一站式供应能力，业务量大幅提升 |

- 数字化转型：AI视觉检测、设备物联、MES国产化替代、Power BI数据运营
- 精益六西格玛持续改进超过20年

### 3. 计量智能终端（30.21亿，19.18%，毛利率39.26%）

**智能电/水/气表 + 能源管理系统**，控股子公司开发科技运营。

| 市场 | 进展 |
|------|------|
| 欧洲 | 西班牙/波兰/荷兰/德国国家级项目 |
| 南美 | 巴西水务数字化突破 |
| 中亚 | 塔吉克斯坦配网改造及计量系统项目 |
| 国内 | 国家电网/南方电网/蒙西电网集采中标1.9亿元 |

- 业务边界从用电侧向配电侧拓展
- 国内超声波水表计量突破，"电水联动"战略落地

## 投资逻辑

**看多**：
1. 存储半导体封测业务受益AI存储需求爆发（+16%高增长）
2. 先进封装布局领先（32D堆叠、Panel FO），技术壁垒不断提升
3. 中国电子实控，国资背景资源优势
4. 三大业务互补：存储半导体（成长）+高端制造（稳定）+计量终端（高毛利）
5. 计量终端海外订单爆发（欧洲多国+巴西），国际化能力强

**看空**：
1. 高端制造毛利率仅9.71%，EMS代工模式天花板明显
2. 经营现金流-27%，应收款或库存可能承压
3. 总资产基本持平（-0.28%），资本开支不足或影响长期增长
4. HDD磁头业务受客户产能转移影响，传统业务面临替代风险

## 风险提示

> [!warning] EMS业务毛利率极低（9.71%）
> 占收入54%的高端制造业务毛利率不到10%，对整体盈利能力形成拖累。

> [!warning] 经营现金流大幅下降（-27%）
> 营收增长但现金流减少，应收款增长或库存积压风险需关注。

> [!warning] 客户集中度高
> EMS模式天然客户集中，前几大客户占收入比重较高。
