# 神工股份（688233.SH）

## 公司概况

锦州神工半导体股份有限公司，中国本土半导体材料和零部件公司，致力于集成电路制造核心工艺材料和零部件的**国产化建设**。全球领先的大直径硅材料制造商，具备"从晶体生长到硅零部件成品"完整制造能力。

## 业务结构

### 1. 大直径硅材料（~46%收入，技术基石）

**集成电路刻蚀用大直径硅材料**，直径覆盖14-22英寸所有规格。

| 指标 | 数据 |
|------|------|
| 产品规格 | 14-22英寸全覆盖 |
| 16寸+占比 | 56.72%（2024年51.61%，持续提升） |
| 16寸+毛利率 | 76.09%（极高盈利能力） |
| 客户 | 中日韩下游刻蚀设备/零部件厂商 |
| 技术地位 | 世界领先水平 |

### 2. 硅零部件（~54%收入，第二增长曲线）

**等离子刻蚀机用硅零部件**，由大直径硅材料精密加工而成，是刻蚀工艺中需**定期更换的核心耗材**。

| 指标 | 数据 |
|------|------|
| 2025收入 | 2.37亿（+100.15%） |
| 占比 | 已超过大直径硅材料 |
| 终端应用 | 存储芯片制造厂等离子刻蚀工艺 |
| 客户 | 中国主流存储芯片制造厂 + 刻蚀设备制造厂 |
| 产能 | 泉州、锦州两地扩产中 |

- 具备"从晶体生长到硅零部件成品"一体化能力
- 已取得中国本土硅零部件市场**领先地位**
- 受益下游客户国产化突破

### 3. 半导体大尺寸硅片（~2%收入，培育期）

- 轻掺低缺陷抛光硅片
- 面向国内主流集成电路制造厂商
- 仍在验证阶段，未来大规模供货基础

## 投资逻辑

**看多**：
1. 硅零部件第二曲线爆发（+100%，占比已超50%），国产替代空间巨大
2. 大直径硅材料16寸+产品毛利率76.09%，极高盈利能力
3. 三年V型反转，半导体周期复苏受益者
4. 一体化制造能力（晶体生长→硅零部件成品），壁垒高
5. 已进入主流存储芯片厂+刻蚀设备厂供应链

**看空**：
1. 体量较小（4.38亿营收），ROE仅5.55%
2. 客户集中度可能较高（中日韩少数大客户）
3. 半导体周期性强，收入波动大
4. 大尺寸硅片业务尚处培育期，贡献极微
