# AI算力与基础设施芯片需求（下游）

## 概述

AI算力军备竞赛是光通信和互连芯片产业链增长的**核心驱动力**。GPU集群对"运力"的需求体现在两个层面：**设备间**远距离光互连（光模块/光引擎）和**设备内**高速电互连（内存接口芯片/PCIe Retimer/CXL）。下游客户高度集中，北美四大云厂商+英伟达主导需求格局。

---

## 下游客户格局

### 核心客户群

| 客户类型 | 代表企业 | 需求特点 |
|----------|---------|---------|
| AI芯片巨头 | **英伟达** | GPU集群光互连，最高规格要求 |
| 北美云厂商 | **Google、Meta、Amazon、Microsoft** | AI训练/推理集群，大规模采购 |
| 通信设备商 | 华为、中兴、思科 | 传统电信/数通市场 |

### 客户集中度数据

| 供应商 | 前五客户占比 | 最大客户占比 | 趋势 |
|--------|------------|------------|------|
| [[中际旭创]] | 76% | 24% | 稳定 |
| [[新易盛]] | <30% | - | **远低于同行**，抗风险最强 |
| [[天孚通信]]（via Fabrinet） | ~90% | 63%（Fabrinet） | 极高且逐年攀升 |

## 需求驱动逻辑

### AI算力 → 光互连需求

```
AI大模型训练/推理 → GPU集群规模扩张 → GPU间高速互连需求 → 光模块/光引擎需求爆发
                                                    ↓
                                            800G → 1.6T → 3.2T 升级周期
```

## 产业链价值流向

### 代工链模式

```
天孚通信（光引擎）→ Fabrinet（代工集成）→ 英伟达（GPU数据中心）
     ↑ 63%收入经此路径
```

### 直供链模式

```
天孚通信（光器件）→ 中际旭创/新易盛（光模块集成）→ 北美云厂商
                                              ↑ 争夺同一批客户
```

## AI算力 → 互连芯片需求

AI服务器不仅需要光互连，还需要大量**设备内电互连芯片**：

| 需求类别 | 芯片类型 | 驱动因素 |
|---------|---------|---------|
| 内存带宽 | RCD/MRCD/MDB（[[澜起科技]]） | 多核CPU/GPU对内存带宽需求迫切 |
| PC端高速内存 | CKD（[[澜起科技]]） | DDR5 6400MT/s+必须配备 |
| PCIe信号完整性 | Retimer（[[澜起科技]]） | AI服务器多GPU互连标配 |
| 内存池化/扩展 | CXL MXC（[[澜起科技]]） | AI推理场景内存需求激增 |

- [[澜起科技]]2025年营收54.56亿（+50%），互连芯片出货量大幅增长
- MRDIMM渗透率提升将带动MRCD/MDB芯片需求爆发式增长
- CXL内存池化在AI推理场景将迎来规模部署

## 下游市场风险

> [!warning] AI资本开支周期性
> 北美云厂商AI投资具有周期性。若AI应用变现不及预期，资本开支可能放缓，光模块需求将受冲击。

> [!warning] 客户集中风险
> [[中际旭创]]前五客户占76%，[[天孚通信]]经Fabrinet占63%。单一客户需求变化对业绩影响巨大。

> [!warning] 地缘政治风险
> 2025年4月以来加征关税/对等关税政策已对光模块行业造成实质影响。[[新易盛]]泰国+越南产能布局可有效分散此风险。

> [!warning] 技术替代风险
> CPO（共封装光学）、LPO（线性直驱）等新技术路线可能改变产业链格局。
