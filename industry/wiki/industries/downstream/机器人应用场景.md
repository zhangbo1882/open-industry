# 机器人应用场景（下游）

## 概述

通用机器人（人形+四足）的下游应用正处于**从科研教育向商业消费、行业应用逐步渗透**的早期阶段。收入结构正在发生显著变化：科研教育占比下降，商业消费和行业应用快速提升。

---

## 各场景详细分析

### 1. 科研教育 — 先导需求，当前最大收入来源

**商业模式**：高校/科研机构采购机器人用于算法研究、二次开发、教学实训

| 指标 | 数据 |
|------|------|
| [[宇树科技]]四足科研教育占比 | 31.58%（2025Q1-3），从68.61%（2022）持续下降 |
| [[宇树科技]]人形科研教育占比 | 73.60%（2025Q1-3），从100%（2023）快速下降 |
| [[优必选]]教育市场份额 | **22.5%，全国第一** |
| 教育机器人行业规模 | 23亿（2022）→ 70亿（2028E），CAGR 20.4% |

**特点**：科研教育是下游应用的先导性需求，是通用机器人大规模商业化的重要基础。客户通过科研教育积累算法和应用经验，为后续商业/行业应用铺路。

### 2. 商业消费 — 短期最大增量

**典型用途**：
- 商业推广（门店/营业厅/景区展示）
- 文旅表演（大型活动、游乐园、景区展演）
- 科技展示（科技馆等机构）
- 家庭陪伴（情感互动、科技潮玩）

| 指标 | 数据 |
|------|------|
| [[宇树科技]]四足商业消费占比 | **42.30%**（2025Q1-3），首次超过科研教育 |
| 线上消费级销售 | 2024年3,037万 → 2025Q1-3 **13,618万**（+348%） |

**特点**：通用机器人作为前沿科技产品，社会各界关注度较高。消费级四足机器人线上销售是增长最快的渠道。

### 3. 工业/行业应用 — 中长期最大空间

#### 3.1 智能巡检

- **电力电网**：浙江500kV兰亭变电站（四足+人形带电作业）
- **石化管道**：巴斯夫湛江石化基地
- **地下管廊**：杭州滨江地下管廊气体检测
- **地铁隧道**：安全巡查

#### 3.2 汽车制造

- [[优必选]] Walker S进入**蔚来第二先进制造基地**（2024.02）
  - 全球首例人形机器人与人类协作完成汽车装配及质量检查
- [[宇树科技]] H1进入蔚来汽车执行搬运、组装

#### 3.3 消防应急

- 火场侦察、气体监测、生命搜救、明火扑灭
- 四足机器人凭借地形适应性强、机动灵活优势突出

#### 3.4 智能制造

- 工厂巡检、搬运、装配
- [[宇树科技]]：中车株洲工厂搬运项目
- 人形机器人执行高重复、高危险、高精密复杂工作

### 4. 家庭服务 — 远期最大市场

**当前阶段**：开发训练期，技术尚未成熟

**主要瓶颈**：
- 家庭环境高度非结构化
- 个性化用户需求远超现有训练数据边界
- 隐私限制加剧数据采集难度

**案例**：
- 中国移动四足机器人狗（陪伴、取快递、安全预警）
- 灵启万物基于G1开发浇花、取快递、家庭清洁功能

## 渗透路径

```
科研教育（当前主力）→ 商业消费（短期增量）→ 行业应用（中期放量）→ 家庭服务（长期愿景）
       ↓                    ↓                    ↓                   ↓
  算法验证/开发      推广/表演/陪伴       巡检/制造/应急       全场景家庭助手
  已规模化收入       快速增长中           验证推广阶段         开发训练阶段
```

---

## 关键瓶颈

> [!important] 大规模商业化仍需突破
> 1. **具身大模型泛化能力**：当前难以覆盖跨场景差异，实时决策能力不足
> 2. **灵巧手精细操作**：触觉感知、控制精度、耐用性均需提升
> 3. **可靠性**：工业/家庭场景需要7×24小时稳定运行
> 4. **成本**：大规模商业化需要进一步降本
