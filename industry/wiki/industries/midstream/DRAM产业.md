# DRAM产业（中游）

## 产业概述

DRAM（动态随机存取存储器）是集成电路产业的战略基础性产品，数字经济时代的核心芯片之一，也是国家信息基础设施战略安全的重要基石。中国是全球主要DRAM消费市场，但**本土品牌自给率极低**，国产替代空间巨大。DRAM行业具有**极高技术门槛、强周期性、高资金密集**三大特征，全球已形成高度垄断格局。

### 中游：DRAM设计+制造（IDM模式）

| 环节 | 说明 |
|------|------|
| 芯片设计 | DRAM电路设计、工艺开发 |
| 晶圆制造 | 12英寸晶圆厂，纳米级制程 |
| 封装测试 | 先进封装（TSV等），委外为主 |
| 模组加工 | DRAM颗粒 → 内存模组组装 |

> DRAM行业采用**IDM模式**（设计+制造一体化），与Fabless模式（如[[澜起科技]]、[[兆易创新]]）不同。IDM模式需要重资产投入（晶圆厂），但能实现工艺与设计的协同优化。

## 全球竞争格局

### 市场份额（2024年，Omdia）

| 厂商 | 全球份额 | 总部 | 特点 |
|------|---------|------|------|
| 三星电子 | **40.35%** | 韩国 | 全球第一，全产业链覆盖 |
| SK海力士 | **33.19%** | 韩国 | HBM领导者，AI存储受益最大 |
| 美光科技 | **20.73%** | 美国 | 全球第三，HBM跟进 |
| **前三合计** | **~94%** | - | **高度垄断** |
| [[长鑫科技]] | **~3.97%** (2025Q2) | 中国合肥 | 中国第一，快速增长 |
| 其他 | ~2-3% | 中国台湾 | 南亚科技、华邦电子等 |

### 行业特征

1. **极高壁垒**：技术门槛+资金门槛+客户认证周期，全球从数十家整合到4家主要厂商
2. **强周期性**：供需波动导致价格大幅波动，2022-2023年深度下行，2024年强劲复苏
3. **规模导向**：规模越大 → 单位成本越低 → 竞争力越强 → 正向循环
4. **高集中度**：前三家占94%，几乎为寡头垄断

## 中国DRAM市场

### 供需矛盾

- 中国是全球**最大的DRAM消费市场**
- 但本土品牌DRAM产品**自给率极低**
- [[长鑫科技]]是**中国大陆唯一实现DRAM大规模量产的企业**

### 国产替代的驱动因素

| 因素 | 说明 |
|------|------|
| 信息安全 | DRAM是国家信息基础设施战略安全的重要基石 |
| 供应链安全 | 地缘政治冲突加速半导体产业链重构 |
| 市场需求 | AI算力、5G、智能汽车驱动DRAM需求持续增长 |
| 政策支持 | 大基金、地方国资持续投入 |

## 与[[互连芯片]]的关系

DRAM存储颗粒与内存接口芯片是**互补配套关系**：

```
DRAM颗粒（长鑫科技/Samsung/SK海力士/Micron）
    + 内存接口芯片 RCD/DB/MRCD/MDB（澜起科技/Rambus）
    + 模组配套芯片 SPD/PMIC/TS
    ↓
内存模组（RDIMM/LRDIMM/MRDIMM）
    ↓
服务器/数据中心
```

- **MRDIMM**是AI服务器的高带宽方案：长鑫DRAM颗粒 + [[澜起科技]]MRCD/MDB芯片
- DRAM技术迭代（DDR4→DDR5，速率提升）推动内存接口芯片同步升级
- 两者需求共同受AI算力驱动，是同一产业链的不同环节

## 市场规模与增长

### 全球DRAM市场规模

- DRAM是半导体最大的细分市场之一
- 受AI服务器需求驱动，HBM（高带宽内存）成为增长最快的细分赛道
- 2024年行业强劲复苏，ASP大幅回升

### 代表性企业收入参考（[[长鑫科技]]）

| 2023 | 90.9 | +9.7% | 行业下行，量增价跌 |
| 2024 | **241.8** | **+166%** | 行业复苏，量价齐升 |

## 关键技术指标

### DRAM代际演进

| 代际 | 速率 | 电压 | 状态 |
|------|------|------|------|
| DDR4 | 最高3200MT/s | 1.2V | 逐步退出 |
| DDR5 Gen1 | 4800MT/s | 1.1V | 主流渗透 |
| DDR5 Gen5 | 8000MT/s | 1.1V | [[长鑫科技]]已量产 |
| LPDDR5X | 最高9600MT/s | - | 移动端主流 |

### 制程工艺

- [[长鑫科技]]已完成第一代到第四代工艺技术平台量产
- 全球头部厂商（三星/SK海力士/美光）已进入10nm级以下制程
- 制程进步直接影响单位成本和存储密度

---

## 发展趋势

1. **短期**：AI算力驱动服务器DRAM需求爆发，HBM成为新增长极
2. **中期**：DDR5全面取代DDR4，MRDIMM渗透率提升拉动DRAM带宽需求
3. **长期**：国产替代持续推进，[[长鑫科技]]份额有望持续提升
