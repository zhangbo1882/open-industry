# 互连芯片产业（中游）

## 产业概述

互连芯片是数据中心和AI基础设施中连接CPU、内存、加速器等核心器件的关键桥梁。随着AI算力规模爆发，"运力"（数据传输能力）已提升至与"算力"（GPU/CPU处理能力）、"存力"（存储容量）并重的战略高度。互连芯片行业正处于**DDR5迭代 + AI需求爆发**的双重驱动期。

---

## 产业细分

### 1. 内存互连芯片

#### 1.1 内存接口芯片（RCD/DB）

| 指标 | 说明 |
|------|------|
| 功能 | 服务器CPU存取内存数据的必由通路 |
| 准入门槛 | 需通过CPU厂商、内存厂商、OEM厂商全方位认证 |
| 竞争格局 | 全球仅3家主要供应商，高度集中 |
| 代际演进 | DDR4(3200MT/s) → DDR5 Gen1(4800) → Gen5(8000MT/s) |

- 每根LRDIMM需要1颗RCD + 10颗DB
- 每根RDIMM需要1颗RCD
- DDR5渗透率持续提升推动需求增长

#### 1.2 高带宽内存接口芯片（MRCD/MDB）

| 指标 | 说明 |
|------|------|
| 功能 | MRDIMM（多路复用双列直插内存模组）核心器件 |
| 代际 | Gen1(8800MT/s) → Gen2(12800MT/s) |
| 特点 | 通过MDB芯片同时访问两个DRAM阵列，实现双倍带宽 |
| 驱动因素 | AI/云计算对内存带宽需求迫切 |

- 每根MRDIMM需1颗MRCD + 10颗MDB + SPD + TS + PMIC
- **MDB芯片需求弹性最大**：MRDIMM渗透率每提升1个百分点，MDB需求大幅增长

#### 1.3 时钟驱动器（CKD）

| 指标 | 说明 |
|------|------|
| 触发条件 | DDR5速率达到6400MT/s及以上时，PC端内存模组**必须配备** |
| 市场 | 台式机（CUDIMM）+ 笔记本（CSODIMM）+ 移动工作站（CAMM/LPCAMM） |
| 代际 | Gen1(7200MT/s) → Gen1+(9200MT/s) |

- 全新市场，DDR5高速化后从0到1的需求创造

### 2. PCIe/CXL互连芯片

| 产品 | 功能 | 成熟度 |
|------|------|--------|
| PCIe Retimer | 解决高速远距离传输信号衰减问题 | PCIe 5.0已产业化 |
| CXL MXC | 内存池化及扩展的核心控制器 | CXL 3.1已送样 |
| PCIe Switch | PCIe信号交换/路由 | 研发中 |

- AI服务器配置多张GPU卡，PCIe Retimer成为标配
- CXL是CPU与加速器/内存之间的高速互连协议，AI推理场景将推动CXL规模部署

### 3. 时钟芯片

- 时钟发生器、时钟缓冲芯片、展频振荡器
- 为电子系统提供精准时钟脉冲信号
- 广泛应用于各类电子系统，不仅限于数据中心

## 关键技术瓶颈

> [!important] JEDEC标准迭代节奏
> 内存接口芯片需紧跟JEDEC标准演进。新一代标准从制定到商用周期较长，需提前布局。

> [!important] SerDes技术壁垒
> PCIe Retimer/CXL产品的核心竞争力来自高速SerDes（串行器/解串器）技术，设计难度高。

> [!important] 生态系统认证
> 新产品需通过CPU厂商（Intel/AMD）、内存厂商（三星/SK海力士/美光）、OEM厂商三方认证，周期长、门槛高。

## 与[[光通信产业链]]的关系

互连芯片与光通信同属AI基础设施"运力"层，但解决不同问题：

| 维度 | 互连芯片 | 光通信 |
|------|---------|--------|
| 解决问题 | 芯片间/设备内高速互连 | 设备间/机房间远距离传输 |
| 核心产品 | RCD/MRCD/PCIe Retimer/CXL | 光模块/光引擎/光器件 |
| 代表企业 | [[澜起科技]] | [[中际旭创]]、[[天孚通信]] |
| 净利润率 | ~41% | 28-47% |
| 共享下游 | AI数据中心 | AI数据中心 |

## 发展趋势

1. **短期（1-2年）**：DDR5渗透+MRDIMM起量+PCIe 5.0 Retimer放量
2. **中期（3-5年）**：CKD进入PC主流、CXL MXC在AI推理场景规模部署、PCIe 6.x/7.0迭代
3. **长期（5-10年）**：以太网PHY/光互连芯片拓展、全互连芯片平台化
