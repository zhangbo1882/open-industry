# 光通信产业链

## 产业链总览

```
上游（原材料/芯片）            中游（光器件/光模块）              下游（终端客户）
┌──────────────────┐     ┌──────────────────────┐     ┌──────────────┐
│ DSP芯片           │     │ 光器件（天孚通信）     │     │ 北美云厂商    │
│ (Broadcom/Marvell)│     │ 光引擎封装            │     │ Google/Meta  │
│ 光芯片            │     │ 无源器件/有源器件      │     │ Amazon/MS    │
│ (Lumentum/Coherent)│───▶│                       │     └──────────────┘
│ 激光器/探测器      │     │ 光模块（中际旭创）     │───▶│ 英伟达        │
│ 陶瓷/金属/玻璃材料 │     │ 全球第一，1.6T量产    │     │ GPU数据中心   │
│ 光纤/电子料       │     │                       │     │              │
└──────────────────┘     │ 代工（Fabrinet）      │     │ 通信设备商    │
                         │ 泰国光模块代工        │     │ 华为/中兴/思科│
                         └──────────────────────┘     └──────────────┘
```

## 上游：光芯片与原材料

### DSP芯片
- **主要供应商**：Broadcom、Marvell
- **壁垒**：全球高度集中，国产替代几乎为零
- **影响**：中际旭创第一大供应商占采购36%，极大概率是Broadcom/Marvell

### 光芯片
- **主要供应商**：Lumentum（全球龙头）、Coherent
- **关键器件**：激光器（VCSEL/DFB/EML）、探测器（PIN/APD）
- **壁垒**：高端光芯片国产化率低，主要依赖美日厂商
- **趋势**：800G/1.6T光模块需求推动高端芯片需求爆发

### 原材料
- 光纤光缆、陶瓷材料（封装基座）、金属材料、玻璃材料、电子料
- 原材料成本占光器件厂商营业成本约75%

## 中游：光器件与光模块

### 光器件（天孚通信、光库科技所在环节）

| 类别 | 代表产品 | 毛利率 | 趋势 |
|------|---------|--------|------|
| 无源器件 | 光纤连接器、波分复用器、光分路器 | ~64% | 稳定增长，技术成熟 |
| 有源器件 | 光引擎（TOSA/ROSA）、光发射/接收组件 | ~47% | 高速增长，AI算力驱动 |

- **核心趋势**：有源器件占比快速提升，光引擎成为AI算力关键组件
- **关键壁垒**：先进封装技术、精密光学加工、客户认证周期长

### 光模块

| 厂商 | 地位 | 营收(2025) | 毛利率 | 核心优势 |
|------|------|----------|--------|---------|
| 中际旭创 | 全球第一 | 382.4亿（+60%） | ~40% | 800G/1.6T量产领先，规模最大 |
| 新易盛 | 全球第三 | 165亿前三季（+222%） | ~47% | 盈利能力最强，客户分散，海外产能 |
| Fabrinet | 全球代工巨头 | — | 10-15% | 英伟达/Google核心代工 |

- **中际旭创**：境外收入占91%，前五客户占76%，AI算力驱动营收高增，1.6T已量产
- **新易盛**：全球排名从第7跃升至第3，毛利率47%/净利率38%行业最高，泰国+越南工厂分散关税风险
- **技术路线**：COB封装、硅光技术、3nm DSP持续迭代；新易盛布局LPO路线降低DSP依赖
- **详细对比**：[[光模块三强对比]]

### 产业链协作模式

**模式一（代工链）**：天孚通信（光引擎）→ Fabrinet（代工集成为光模块）→ 英伟达（终端数据中心）

**模式二（直供链）**：天孚通信（光器件）→ 中际旭创/新易盛（集成为光模块）→ 北美云厂商（Google/Meta/Amazon/MS）

中际旭创和新易盛同为光模块厂商，共享天孚通信（上游光器件）和博通（DSP芯片）等上游供应商，争夺同一批北美云厂商客户。详见 [[光模块三强对比]]。

**模式三（器件直供链）**：光库科技（光通讯器件/铌酸锂调制器）→ 中际旭创/新易盛/北美云厂商（光模块集成）。光库科技同时向光纤激光器和激光雷达市场供货，业务多元化。

## 下游：终端客户

### AI数据中心（核心增长动力）
- **英伟达**：GPU集群需要大量高速光互连
- **Google/Meta/Amazon/Microsoft**：AI训练/推理集群光模块需求
- **驱动因素**：800G→1.6T升级周期，AI算力军备竞赛

### 通信设备商
- 华为、中兴通讯、思科等传统电信/数通设备商

## 产业链财务梯度

| 环节 | 代表公司 | 营收(2025) | 毛利率 | 特点 |
|------|---------|----------|--------|------|
| DSP芯片 | Broadcom/Marvell | — | 60-70% | 极高壁垒，高度垄断 |
| 光芯片 | Lumentum | — | 30-40% | 高技术壁垒，周期性强 |
| 光器件 | 天孚通信 | 51.6亿 | 53.6% | 高盈利，客户集中度高 |
| 光器件 | 光库科技 | 14.7亿 | 34.66% | TFLN调制器+多元光器件，光纤激光基本盘 |
| 光模块 | 中际旭创 | 382.4亿 | ~40% | 规模效应，竞争激烈 |
| 代工 | Fabrinet | — | 10-15% | 重资产，依赖大客户 |

## 核心关注点

1. **AI算力驱动**：英伟达GPU集群是光通信产业链核心增长引擎
2. **技术升级周期**：400G→800G→1.6T，每次升级带来量价齐升
3. **国产替代**：高端光芯片国产化是长期趋势
4. **客户集中风险**：产业链各环节均面临大客户集中问题
5. **地缘政治风险**：光芯片/DSP芯片受制于美日，贸易摩擦影响供应链。2025年4月以来加征关税已对光模块行业造成影响
6. **产业链上下游对比**：光器件（天孚通信，毛利率53.6%）盈利能力显著高于光模块（中际旭创，毛利率约40%），但光模块体量远大于光器件
7. **薄膜铌酸锂（TFLN）调制器**：1.6T以上超高速光模块核心技术方案，光库科技是国内稀缺的TFLN产业化能力厂商，2025年部分产品已批量出货
