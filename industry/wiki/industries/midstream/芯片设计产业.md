# 芯片设计产业（IC Design）

## 产业定位

芯片设计处于半导体产业链的中游（设计→制造→封测），是具有自主知识产权并体现核心技术含量的环节。Fabless模式下，设计公司专注研发和销售，将制造/封测外包给代工厂。

```
上游（设备/材料）      中游（设计→制造→封测）         下游（终端应用）
┌──────────────┐   ┌────────────────────────┐   ┌──────────────┐
│ 半导体设备     │   │ 芯片设计（Fabless）      │   │ 消费电子      │
│ (ASML/TEL等)  │   │ 兆易创新/澜起科技/寒武纪 │   │ 汽车电子      │
│ 硅片/光刻胶    │──▶│                         │──▶│ 工业控制      │
│ 电子气体/靶材  │   │ 晶圆代工（Foundry）      │   │ 通讯/网络     │
│               │   │ 台积电/中芯国际/华虹     │   │ 服务器/数据中心│
└──────────────┘   │                         │   │ 物联网        │
                   │ 封装测试（OSAT）         │   └──────────────┘
                   │ 长电科技/通富微电/华天   │
                   └────────────────────────┘
```

## 产业规模

- **全球半导体市场**（2025）：7,720亿美元（+22%）
- **中国IC设计收入**（2025）：约1,180亿美元（+29.4%）
- **细分市场规模**：
  - 专用型存储（NOR Flash + SLC NAND + 利基DRAM）：157亿美元
  - MCU：210亿美元
  - 模拟芯片：830.9亿美元

## 主要细分赛道

### 存储芯片

| 类型 | 2025年市场规模 | 主要玩家 | 国产化率 |
|------|-------------|---------|---------|
| NOR Flash | 31亿美元 | 华邦/旺宏/兆易创新(全球第二) | 较高 |
| SLC NAND | 27亿美元 | 兆易创新/华邦/旺宏 | 中等 |
| 利基型DRAM | 99亿美元 | 兆易创新/南亚科/华邦 | 较低 |
| 主流DRAM | — | 三星/SK海力士/美光（三大寡头） | 极低 |
| NAND Flash | — | 三星/SK海力士/美光/铠侠/西部数据 | 极低 |

- **行业趋势**：海外大厂淡出利基型存储→供给收缩→利基产品量价齐升
- **AI驱动**：端侧AI对代码存储（NOR Flash）需求增加

### MCU（微控制器）

| 梯队 | 代表公司 | 特点 |
|------|---------|------|
| 全球龙头 | 意法半导体/恩智浦/英飞凌/微芯 | 车规/工业级优势 |
| 国内龙头 | 兆易创新（32位MCU国内第一） | 消费/工业为主，向车规突破 |
| 国内第二梯队 | 华大半导体/中颖电子/国民技术 | 各有细分领域优势 |

- **行业趋势**：价格处于周期底部，库存水位已低，等待复苏信号
- **兆易创新突破方向**：车规MCU（GD32A系列）、高算力MCU（GD32H7）、AI MCU

### 互连芯片/内存接口

详见 [[互连芯片]]

- **代表公司**：[[澜起科技]]（内存接口芯片全球前三）
- **下游**：数据中心/服务器，直接受益AI算力爆发

### 模拟芯片

- **市场规模**：830.9亿美元（2025年预测）
- **代表公司**（海外）：TI、ADI、Infineon
- **代表公司**（国内）：圣邦股份、思瑞浦、兆易创新（收购苏州赛芯后切入）
- **国产化率**：极低（~15%），国产替代空间巨大

## 产业链财务梯度

| 环节 | 代表公司 | 毛利率 | 特点 |
|------|---------|--------|------|
| EDA/IP | Synopsys/ARM | 80-90% | 极高壁垒，垄断 |
| 芯片设计 | 兆易创新/澜起科技 | 35-45% | 知识产权核心，轻资产 |
| 晶圆代工 | 台积电/中芯国际 | 25-55% | 重资产，规模效应 |
| 封装测试 | 长电科技/通富微电 | 15-25% | 劳动密集，竞争激烈 |

## 核心关注点

1. **AI驱动**：端侧AI（AI手机/AIPC/机器人）推动存储和MCU需求升级
2. **国产替代**：美国制裁倒逼国产化，IC设计是国产替代最活跃的环节
3. **存储周期**：利基型存储受益海外大厂退出，周期向上
4. **汽车电子**：车规MCU/存储是长期增长驱动力，认证周期长但粘性高
5. **Fabless模式风险**：依赖代工厂产能，地缘政治影响供应链安全
