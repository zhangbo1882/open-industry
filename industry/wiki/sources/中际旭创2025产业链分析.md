# 中际旭创（300308）2025年产业链分析 — 资料摘要

> 数据来源：中际旭创2025年年度报告全文（巨潮资讯网，2026-03-31发布）
> 分析日期：2026-04-18

## 核心财务数据

- **营收**：382.40亿元（同比+60.25%）
- **净利润**：107.97亿元（同比+108.78%）
- **光通信收发模块**占营收97.95%
- **境外收入**占比90.58%
- **直销模式**占比98.69%

## 行业地位

- 2023-2024年连续位居Lightcounting全球光模块厂商**第一名**
- 产品覆盖100G至1.6T全系列光模块
- 1.6T产品已量产交付

## 产业链定位

```
DSP芯片(Broadcom/Marvell) + 光芯片(Lumentum等) → 中际旭创(光模块全球第一) → 北美云厂商(Google/Meta/Amazon/MS)
```

## 核心风险

1. 客户集中（前五占76%，客户A占24%）
2. 供应链安全（供应商A独占36%，海外芯片厂商）
3. 贸易政策风险（境外收入91%，关键原材料海外采购）
4. 2025年4月以来加征关税/对等关税政策影响

## 技术路线

- COB封装（业内率先使用）
- 硅光技术（2022年展示800G硅光模块，2024年展示硅光800G ZR OSFP相干模块）
- 3nm DSP（2025年OFC展示3nm 1.6T OSFP 2xFR4光模块）
