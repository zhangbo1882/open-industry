## 核心观点

1. 中国人工智能赋能机器人的头部企业，产品涵盖教育、物流、消费级机器人及解决方案
2. 2022年在中国智能服务机器人产业排名第三，教育细分领域排名第一
3. 2024年2月Walker S进入蔚来工厂，全球首例人形机器人与人类协作完成汽车装配及质量检查
4. 预期三年内人形机器人将率先在汽车工厂实现应用落地和持续放量

**风险提示**：人形机器人技术/产业化发展不及预期、下游需求不及预期、核心技术泄露风险、行业竞争加剧等

## 公司业务结构

### 1. 教育机器人（市场份额22.5%，全国第一）

| 指标 | 数据 |
|------|------|
| 行业规模(2022) | 23亿元 |
| 行业规模(2028E) | 70亿元 |
| 行业CAGR(2022-2028) | 20.4% |
| 市场渗透率(2022) | 仅8%，空间广阔 |
| 公司份额 | 22.5%（TOP 1） |

- 产品：Alpha Mini悟空（教育）、Cruzr、Yanshee、uKit
- 分为基础教育（K12）和高校教育两大类
- TOP 5企业合计市占率32.9%（优必选、盛通、鲸鱼机器人、大疆创新、童心制物）

### 2. 物流机器人

| 指标 | 数据 |
|------|------|
| 行业CAGR(2022-2028) | 30.4% |
| 公司排名 | 第7 |
| 公司份额 | 2.2% |

- 产品：瓦力系列机器人（AGV/AMR）、自动化仓储系统（AS/RS）
- TOP 5：海康机器人、极智嘉科技、东杰智能、井松科技、海柔创新

### 3. 人形机器人（国内率先商业化）

- Walker（2018首发）→ Walker 2nd gen（2019，春晚表演）→ Walker X（2021，2022商业化）→ Walker S（2024进入蔚来工厂）
- Walker X：41个自由度，7自由度手臂+6自由度手掌，多维力觉+多目立体视觉+全向听觉
- 中国首个商业化的双足真人尺寸人形机器人

### 4. 其他方向

- **巡检机器人**：室内外多环境适用
- **康养机器人**：满足老年人需求，提升康养机构服务质量
- **消费级机器人**：Alpha Mini（非教育）、积木系列、AiRROBO猫砂机、AiRROBO扫地机器人

## 公司核心竞争力

1. **产品丰富**：50+种产品，累计售出超76万台机器人
2. **研发实力**：717名研发人员（2023.6），1800+注册专利（380+海外专利），全栈式技术布局
3. **技术体系**：机器人技术 + AI技术 + 机器人AI融合技术 + ROSA系统
4. **品牌力**：国内领先，2023H1海外营收占比26.7%
