## 产品矩阵

### 互连类芯片（核心业务）

#### 1. 内存互连芯片

| 产品 | 功能 | 最新代际 | 速率 |
|------|------|---------|------|
| RCD芯片 | 服务器内存模组核心逻辑器件 | DDR5 Gen5.0 | 8000MT/s |
| DB芯片 | 数据缓冲器（与RCD套片） | DDR5 Gen1.0 | 4800MT/s |
| MRCD芯片 | 高带宽内存模组MRDIMM专用 | DDR5 Gen2.0 | 12800MT/s |
| MDB芯片 | MRDIMM数据缓冲器 | DDR5 Gen2.0 | 12800MT/s |
| CKD芯片 | PC端内存模组时钟驱动器 | DDR5 Gen1.0+ | 9200MT/s |
| SPD/PMIC/TS | 内存模组配套芯片 | - | - |

- DDR5 Gen5 RCD芯片速率是DDR4最高速率的2.5倍
- DDR5 Gen5 RCD同速率下功耗较Gen1下降近40%
- 每根MRDIMM需搭配1颗MRCD + 10颗MDB + SPD/TS/PMIC

#### 2. PCIe/CXL互连芯片

| 产品 | 进展 |
|------|------|
| PCIe 5.0 Retimer | 已产业化 |
| PCIe 6.x/CXL 3.x Retimer | 已发布，含AEC方案 |
| PCIe 7.0 Retimer | 研发中 |
| PCIe Switch | 研发中 |
| CXL MXC（内存扩展控制器） | 全球首发，CXL 3.1版已向主要客户送样 |

#### 3. 时钟芯片

- 时钟发生器芯片、时钟缓冲芯片、展频振荡器
- 广泛应用于各类电子系统

### 津逮®产品线

- 津逮®CPU（x86架构，含PC/DSC安全功能）
- 数据保护和可信计算加速芯片

## 核心竞争力

1. **内存互连芯片全球领先**：DDR2-DDR5全系列覆盖，牵头JEDEC国际标准制定
2. **CXL全球首发**：MXC芯片为全球首款内存扩展控制器
3. **自研SerDes技术**：支撑PCIe Retimer全线产品
4. **高研发投入**：583名研发人员（占员工74.4%），其中硕士及以上64%
5. **Fabless模式**：专注设计，制造委托专业代工厂

---

## 增长驱动因素

- DDR5渗透率持续提升（从DDR4向DDR5迁移）
- AI产业爆发推动内存互连芯片需求大幅增长
- MRDIMM（高带宽内存模组）渗透率提升 → MRCD/MDB需求爆发
- CKD芯片进入PC市场（DDR5 6400MT/s以上必须使用）
- PCIe Retimer需求随AI服务器规模扩张而增长
- CXL内存池化在AI推理场景加速落地
